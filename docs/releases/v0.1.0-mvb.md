## What shipped

Sentinel's Minimum Viable Brain v0.1 — a Financial AGI core that
**observes, audits, and never acts**. The end-to-end pipeline
runs a synthetic ObservationEvent through compile → workspace
pulse → recall-trigger check → optional deontic gate, and produces
`SystemOutput.WAIT` with a fully chained M1 audit trail.

```
Phase 1   Contracts as Code            (8 schema + 2 constitution modules)
Phase 2   Observer Ledger              (catalog + hash chain + JSONL +
                                         permanence + ring buffer +
                                         audit reader + router)
Phase 3   Numerics Loader              (dev_only guard + 8 dev fixtures +
                                         dependency validator)
Phase 4   Ingress Compiler             (profile caps + soft overlap +
                                         rules + compiler)
Phase 5   Minimal M0 Tissue            (neuron + synapse + self_field +
                                         proto_resonance + assembly +
                                         tissue)
Phase 6   Workspace Pulse              (single mechanism)
Phase 7   Gates                        (deontic — 11 declaratives;
                                         memory write — silent,
                                         candidate-only)
Phase 8   Recall Skeleton              (composite-AND trigger + top-1
                                         + candidate guard + audit)
Phase 9   EchoAdapter                  (signed mock, observe-only,
                                         multiplicative trust)
Phase 10  End-to-end Dry Simulation    (runtime/dry_sim + CLI +
                                         integration tests +
                                         forbidden-output guard)
Polish    Audit wiring + property tests + public API + catalog
          consistency guard + permanence-aware router
```

## Explicitly forbidden in v0.1

```
- Live exchange / venue API integration
  (ccxt, binance, btcturk, bybit, okx, gate_api, kucoin, huobi,
   bitfinex, kraken, web3) - CI-enforced grep
- LLM SDK integration
  (openai, anthropic, langchain) - CI-enforced grep
- Execution-output literals
  ("BUY", "SELL", "EXECUTE_REAL", "ORDER_SUBMIT" — never in
   sentinel/*.py, never in any M1 ledger payload reason)
- M2 verified-status memory production
  (mvp_verified_disabled = True; candidate-only writes)
- Numerics runtime mutation
  (signed artifacts only; loader rejects dev_only=True in
   PRODUCTION mode)
- Cross-instance / fork / migration paths
- Replay-driven memory update
  (mvp_replay_enabled = false)
- M0 learning
  (learning_enabled = false; synapse weights immutable)
- Silent action
  (every deontic / memory / adapter / recall decision is audited
   to M1)
- Adapter neural_seed emission
  (constitutional red line; raises InvariantViolation +
   ADAPTER_MANIFEST_STATUS_CHANGED reason=neural_seed_emission_attempt)
- Operator override of deontic gate
- Human/LLM-direct recall push (top-1, core-trigger only)
```

## Dry simulation pipeline

```
EchoAdapter activation         -> ADAPTER_MANIFEST_STATUS_CHANGED  (PERMANENT -> JSONL)
ObservationEvent emitted        (in-memory only; not audited)
OBSERVATION_INGESTED            (RING_BUFFER_ONLY -> ring buffer or drop)
compile_neural_seed            (pure; no audit)
WORKSPACE_PULSE                 (RING_BUFFER_ONLY -> ring buffer or drop)
recall trigger check           -> RECALL_TRIGGER_REJECTED         (PERMANENT -> JSONL)
(optional) ApprovedActionIntent -> DEONTIC_BLOCKED                (PERMANENT -> JSONL)
SystemOutput.WAIT
```

Per-canonical-run audit count:
    2 PERMANENT events (adapter activation + recall rejected)
    2 RING_BUFFER_ONLY events (observation + pulse)
With `exercise_deontic_gate=True`: +1 PERMANENT (DEONTIC_BLOCKED).

## How to run locally

```bash
# Bağımlılıkları kur
uv sync

# Test + tip + lint
uv run pytest
uv run pyright
uv run ruff check .
uv run ruff format --check .

# CLI dry simulation
uv run python -m sentinel.runtime --ledger /tmp/sentinel-mvb.jsonl
# -> sentinel-mvp: output=WAIT audit_events=4 permanent=2 ring=0 ledger=...

# Programmatic
python -c "
from sentinel import EchoAdapter, JsonlObserverLedger, run_dry_simulation
ledger = JsonlObserverLedger('/tmp/run.jsonl')
result = run_dry_simulation(
    ledger=ledger,
    adapter=EchoAdapter.default(),
    observation_magnitude=0.8,
)
print(result.output)         # SystemOutput.WAIT
print(ledger.verify())       # True
"
```
